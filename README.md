# 摸鱼岛摸鱼插件（Chrome 扩展）

在浏览器中定时调用接口、自动抢摸鱼岛聊天室红包，并查看相关日志。

## 安装与加载

1. 打开 Chrome（或 Edge），地址栏输入 `chrome://extensions`（Edge 为 `edge://extensions`）。
2. 打开右上角 **「开发者模式」**。
3. 点击 **「加载已解压的扩展程序」**，选择本项目的 **`api_plugin` 文件夹**（包含 `manifest.json` 的目录）。
4. 工具栏出现图标后，点击即可打开弹窗。

更新代码后，在扩展管理页对该扩展点击 **「重新加载」**。

## 界面说明

弹窗顶部为 **「配置」** 与 **「日志」** 两个标签页。

### 配置

**定时 API 任务**

- 填写任务名称、API 地址（`http`/`https`）、请求方法（GET/POST）、调用间隔（秒）。
- 点击 **「新增任务」** 保存；后台会按间隔自动请求，并在请求头中带上你在下方保存的 **Token**（`Authorization: Bearer <token>`）。
- 任务列表中可对单条任务点击 **「删除」**。

**Token 与抢红包**

- **Token**：与摸鱼岛 / 接口站一致的鉴权串；可手动粘贴后点 **「保存 Token」**，或在已登录摸鱼岛网页的标签页上点 **「自动获取 Token」**（会读取该页 `localStorage` 中的 `tokenValue`）。
- **开启自动抢红包**：总开关；关闭后不会发起抢红包请求。
- **插件内直连 WS 抢红包**：由扩展后台直接连接 WebSocket，收到红包相关推送后自动调用抢红包接口。勾选后会 **立即写入存储**（无需先点保存）。
- **聊天室 WS 模板**：例如 `wss://api.yucoder.cn/ws/?token={token}`，其中 `{token}` 会替换为已保存的 Token。
- **抢红包 API 基础地址**：例如 `https://api.yucoder.cn`，抢红包请求发往该域名下的 `/api/redpacket/grab`。
- 修改 WS 或 API 地址后，点击 **「保存抢红包与 WS 设置」**；校验通过后会写入并触发后台重连 WS。

### 日志

自上而下依次为：

1. **聊天室 WS**：展示后台记录的聊天室类 WebSocket 消息摘要（格式为 **名字：内容**；红包显示 **【红包】**，图片类显示 **【图片】**）。点 **「刷新」** 从本地存储重新渲染。
2. **抢红包 · 按日统计**：按本地日期汇总成功次数与积分合计。
3. **抢红包 · 明细**：每次抢红包接口返回的摘要（成功时会显示抢到多少积分）。
4. **接口调用**：定时任务与抢红包相关 HTTP 调用的简要记录。

底部绿色/红色文案为操作状态提示。

## 抢红包的两种数据来源

1. **插件内直连 WS**（推荐与「配置」中开关配合）：后台 Service Worker 连接你配置的 WS，解析消息中的红包 ID 并抢红包。
2. **网页内 WebSocket**：扩展会向所有 `http/https` 页面注入脚本，对页面中的 `WebSocket` 进行监听；若消息中出现 `[redpacket]...[/redpacket]` 标记，会向后台上报红包 ID 并尝试抢红包。你在摸鱼岛网页里正常打开聊天室即可触发（无需在弹窗里再填聊天室地址）。

抢红包接口始终使用 **「抢红包 API 基础地址」** 中的域名，不会误用普通网页的 `origin`。

## 权限说明（manifest 摘要）

- **alarms**：定时调度 API 任务。
- **storage**：保存 Token、任务、日志等。
- **scripting**：从当前标签页读取 `tokenValue`。
- **tabs**：在需要时打开新标签页（若后续功能使用）。
- **host_permissions**：对配置的 API 与 WS 发起网络请求；内容脚本注入任意 `http/https` 页以监听 WS。

## 注意事项

- Token 与接口域名请与正式环境一致；勿将含 Token 的截图或仓库公开提交。
- 日志与统计保存在本机 **chrome.storage.local**，清除浏览器数据或卸载扩展会丢失。
- 若抢红包或 WS 异常，可先查看 **日志** 页中的明细与 **接口调用**，并确认 Token 未过期、WS 模板与 API 地址正确。

## 项目结构（简要）

| 文件 | 作用 |
|------|------|
| `manifest.json` | 扩展清单 |
| `background.js` | 定时任务、WS 连接、抢红包、写日志 |
| `popup.html` / `popup.js` | 弹窗界面与交互 |
| `content.js` / `ws-hook.js` | 页面内 WS 桥接，发现红包 ID |
| `icons/fishpi.png` | 扩展图标 |
